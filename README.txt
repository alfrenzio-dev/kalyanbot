Demo Telegram WebApp — Orders
Files:
- index.html : main page
- style.css : styles
- app.js : client-side logic (simulates API via localStorage)

How to run:
1. Распакуйте OrdersWebApp_Demo.zip
2. Откройте index.html в браузере (в офлайн-режиме работает)
3. Вкладка 'Активные' показывает активные заказы.
4. Нажмите 'Добавить в заказ' чтобы открыть модал и добавить товары.
5. 'Завершить' помечает заказ как completed.
